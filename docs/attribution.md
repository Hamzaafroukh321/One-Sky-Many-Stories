# Attribution Notes

The app must include visible in-product attribution for all sky data and cultural story sources.

## Required In-App Attribution

> Star positions: HYG Database by David Nash / astronexus (CC BY-SA 4.0). Constellation figures & myths: Stellarium Sky Cultures and the constellation-lines dataset by Doina Bucur (Creative Commons). This project is non-commercial and educational.

## Data Sources

- HYG Database: https://codeberg.org/astronexus/hyg
- HYG Database mirror: https://github.com/astronexus/HYG-Database
- Stellarium Sky Cultures: https://github.com/Stellarium/stellarium-skycultures
- Constellation Lines: https://github.com/doinab/constellation-lines

## Implementation Notes

- Keep attribution visible in the About / Sources view.
- Preserve culture-specific license details where available.
- Do not ship generated data without confirming the license for each included culture.
- HYG is CC BY-SA 4.0, so downstream usage must respect attribution and share-alike requirements.
